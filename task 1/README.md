# CodeAlpha Simple Storage Smart Contract

A beginner-friendly Solidity smart contract developed as part of the CodeAlpha Blockchain Development Internship.

## 📌 Project Overview

This project demonstrates the basic functionality of a smart contract using Solidity. The contract allows users to store and manage an integer value on the blockchain.

The project was built and tested using Remix IDE and deployed on the Ethereum Virtual Machine (EVM).

---


## 🚀 Features

- Store integer values on-chain
- Increment stored value
- Decrement stored value
- Set custom values
- Publicly read stored data
- Event emission for value updates

---

## 🛠 Technologies Used

- Solidity
- Remix IDE
- Ethereum Virtual Machine (EVM)

---

## ▶️ How to Run

1. Open Remix IDE
2. Create a Solidity file
3. Paste the smart contract code
4. Compile using Solidity version `0.8.20`
5. Deploy using Remix VM
6. Test the contract functions
